# 폭포수 

# 반복점증적 모델
## 개념 
* 불완전한 요구사항으로 초기버전을 만든 후, 요구사항을 정제하는 작업을 반복하여 만들어가는 소프트웨어 프로세스 모델
* 사용자의 요구사항 또는 제품의 일부분을 점증적으로 병행 개발 후 통합하는 방법

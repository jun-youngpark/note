## 재고 시스템으로 알아보는 동시성 이슈

homebrew에서 도커설치하기
- brew install docker

mysql 설치 및 실행
- docker pull mysql
- docker run -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=1234 --name mysql mysql 
- docker ps (프로세스 확인)

my sql 데이터베이스 생성
- docker exec -it mysql bash
- mysql -u root -p
- create database stock_example;
- use stock_example;
